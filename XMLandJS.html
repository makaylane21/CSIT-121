What is JavaScript? How is JavaScript related to HTML and CSS? What are some common JavaScript elements, properties, methods, and functions?
JavaScript is type of programming language that is used to make webpages interactive. While HTML is used to create and give structure to a websites contents, CSS adds style to the structures. JavaScript allows the structures that were styled to become interactive with the users who are on the website. They all work together to give a user the best experience possible. A common element is the script tag that can be embedded in HTML. Methods of JavaScript include but are not limited to alert messages, popup windows, and jump menus. An example of a JS property is a named data property. An example of a function is an alert message popping up when you are on a website.
